# Project Setup 
1. git clone https://api_token@github.com/vdurshetty/file-upload.git
2. python -m venv file-upload   (will pick the latest python version) [for specific version use # python3.10 -m venv file-upload]
3. source file-upload/bin/activate
4. cd file-upload 
5. create requirements.txt 


## Install python libraries 
% pip install -r requirements.txt 
% pip uninstall -r requirements.txt 

## Run the REST API server 
% gunicorn main:app
